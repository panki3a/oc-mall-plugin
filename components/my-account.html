# MyAccount

The MyAccount component displays an overview of the user's account.

It pulls together the OrdersList, CustomerProfile and AddressList components.

By changing the page property a different child component is displayed.

# Properties

# page (string)

The currently active page. This property should be populated via URL parameters. Possible values are orders, profile and addresses.

# Example implementations

# Display the my account page

title = "Account"
url = "/account/:page?"

[session]
security = "user"
redirect = "login"

[myAccount]
page = "{{ :page }}"
